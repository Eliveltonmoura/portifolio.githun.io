Sobre Mim

Olá! Meu nome é Elivelton e sou apaixonado por tecnologia, desenvolvimento web e sistemas embarcados. Atualmente, curso Emgenharia da Computação na Universidade Federal do Ceará e tenho experiência com JavaScript, TypeScript, Node.js, React, mySQL, C , C++ e PHP. Projeto 2

Este projeto é uma réplica simplificada do Twitter, desenvolvida com HTML, CSS e TypeScript. Ele permite que os usuários realizem login, criem postagens e visualizar seus perfis. A interface foi projetada para ser intuitiva e responsiva, proporcionando uma experiência semelhante à da plataforma original.

O objetivo do projeto foi aprimorar habilidades em desenvolvimento frontend e tipagem com TypeScript, além de explorar conceitos como manipulação de estado e estruturação de código em aplicações web..
